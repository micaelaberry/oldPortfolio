Projects

Here is a list of some fun projects I have worked on over the past few weeks. Click on the title of each project to be directed to the GitHub repository.

Virtual Pets AMOK!



Our virtual pets at the shelter have run amok! But no worries because the Doctor is here and he is here to help. The theme of this project was Doctor Who and all the pets are the companions. This app has multiple classes where methods are called upon to feed, water, oil, walk, and clean the cages of our pets. We have not only organic pets but robotic pets as well. Allons-y! The IDE used for this project was Eclipse and the Java skills required were:
  • Implenting a Game and Tick loop
  • Introducing new methods for robotic pets
  • Return a Collection of pets
  • Constructors
  • Introduce new attributes for the health of the pets
  • Create new models for organic and robotic pets



Virtual Pet Shelter App



Time to volunteer at the pet shelter! With this multi-class app, you can add, adopt, feed, water, and play with all the pets inside. I used a pirate theme for my shelter so the pets could also pillage and sail the 7 seas. Arrrrrrgh! The IDE used for this project was Eclipse and the Java skills required were:
  • Implenting a Game and Tick loop
  • Ask for user input
  • Return a Collection of pets
  • Constructors
  • Add, remove pets from user input
  • Instance Variables



Fortune Teller App



By entering the magical world of Mystical Mickey, you will have your fortune told by answering a few simple questions before being amazed! The IDE used for this project was Eclipse and the Java skills required were:
  • Variables and Basic Types
  • Operators and Expressions
  • Asking for user input
  • Conditionals
  • Strings
